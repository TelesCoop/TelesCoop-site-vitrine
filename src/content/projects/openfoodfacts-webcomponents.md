Openfoodfacts Webcomponents est une suite de webcomponents pour intégrer les éléments d'Openfoodfacts dans vos sites. Il permet d'afficher visuellement des retours d'outils de Machine learning pour améliorer la qualité des donnés, de composants utiles sur plusieurs sites, tel que le scanner de code barre.
